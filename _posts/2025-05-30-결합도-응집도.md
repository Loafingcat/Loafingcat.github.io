# 결합도

1. 내용 결합도

내부 기능 및 내부 자료를 직접 참조

2. 공통 결합도

전역 변수

3. 외부 결합도

외부의 다른 모듈에서 참조

4. 제어 결합도

논리적 흐름 제어, 제어 신호, 제어 요소 전달

5. 스탬프 결합도

배열이나 레코드 등 자료구조 전달

6. 자료 결합도

자료 요소로만


# 응집도

1. 기능적 응집도

모든 기능 요소가 단일 문제와 연관

2. 순차적 응집도

출력 데이터가 다음 활동의 입력 데이터

3. 교환(통신)적 응집도

동일한 입출력 다른 기능

4. 절차적 응집도

다수의 기능이 순차적으로 수행

5. 시간적 응집도

특정 시간에 처리되는 기능들 묶음

6. 논리적 응집도

유사한 성격이나 특정 형태로 분류되는 처리 요소들 묶음

7. 우연적 응집도 

관련 없는 것들로 구성
